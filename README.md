# 5GHz Cross-Coupled LC-Tank Oscillator-SiGe-Design
Design and simulation of a 5GHz cross-coupled LC-tank RF oscillator using 0.25µm SiGe technology. Developed in Keysight ADS with a focus on phase noise optimization and power consumption.


## Project Overview
This repository contains the design and simulation of a high-frequency **LC-Tank Oscillator** operating at 5GHz. The project was developed using **0.25µm SiGe (Silicon-Germanium)** technology, focusing on frequency stability and the startup conditions for high-frequency resonators.

## Technical Specifications
* **Operating Frequency:** 5.0 GHz
* **Topology:** Cross-coupled LC-tank for differential output.
* **Technology:** 0.25µm SiGe BiCMOS process.
* **Simulation Tool:** Keysight ADS (Advanced Design System).

## Key Analysis Performed
* **Startup Analysis:** Evaluation of the negative resistance ($G_m$) required to overcome tank losses.
* **Harmonic Balance (HB):** Accurate determination of the steady-state oscillation frequency and output power.
* **Phase Noise Performance:** Characterization of the spectral purity of the 5GHz signal.
